# СП Кошелёк

С этим приложением вы можете переводите деньги другим игрокам и между своими счетами, а также создавать отдельные наличные счета и всегда видеть свой общий баланс.

### Основные возможности
- Денежные переводы другим игрокам
- Денежные переводы между собственными счетами
- Создание собственных счетов для учёта наличных средств
- Подсчёт общего баланса со всех счетов


### **Регистрация**
1. Необходимо приобрести проходку на один из серверов [СП](https://spworlds.ru/)
2. На сайте [СП](https://spworlds.ru/) зайдите во вкладку "Кошелёк", выберете карту и нажмите иконку "Поделится"
3. Зайдите на сервер, для которого хотите активировать крату, и сгенерируйте для неё id и token
4. Введите данные карты в приложение. Данные хранятся только на вашем устройстве
