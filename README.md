# SP Wallet — приложение для управления игровой валютой

С этим приложением вы можете переводите деньги другим игрокам и между своими счетами, создаёте отдельные наличные счета и всегда видите общий баланс. Все данные карт хранятся только на вашем устройстве. Для регистрации карт нужна проходка на один из серверов СП (https://spworlds.ru/).

Основные возможности включают переводы игрокам и между своими картами, создание наличных кошельков и подсчёт общего баланса со всех счетов.
