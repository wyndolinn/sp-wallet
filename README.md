Это приложение помогает быстро переводить деньги другим игрокам и между своими счетами, а также создавать отдельные наличные счета для удобного учёта. Данные карт хранятся только на вашем устройстве, так что они в безопасности.
Для регистрации нужна проходка на один из серверов СП (https://spworlds.ru/), а дальше всё делается прямо в приложении.
